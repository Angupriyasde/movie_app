# README

* Clone the project

    `git clone https://github.com/Angupriyasde/movie_app.git`

* Install rails

    `gem install rails`

* Install gems

    `bundle install`

* Migrate DB

    `rails db:migrate`

* Start server

    `rails server`

* Hit the url
    
    `http://localhost:3000/`
